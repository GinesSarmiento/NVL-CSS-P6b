Práctica de selctores (II)

En este ejercicio se van a aplicar reglas de parentesco a diferentes selectores.

La regla número 1(r1) determina que los p hijos directos de un div tienen que tener un borde de color fabada.

La regla número 2(r2) determina que los primeros elementos p de un div deben llevar la clase "importante".

La regla número 3(r3) determina que los elementos p que están situados al lado de los párrafos de clase "importante" tendrán el texto de color rojo.

Aquí empieza el ejercicio

Este párrafo tiene una clase "importante" y es hijo directo de un div. Cumple las reglas r1 y r2.

Este párrafo es hijo directo de un div y está al lado de un párrafo con clase "importante". Cumple las reglas r1 y r3.

Este título pertenece a un elemento span.

Este párrafo es hijo de un elemento span y nieto de un div, no cumple ninguna regla.

Este párrafo tiene una clase "importante" y es hijo directo de un div. Cumple las reglas r1 y r2.

Este título está situado al lado de un párrafo de clase "importante", pero no es un párrafo, por lo que no cumple ningúna regla.

Este párrafo es hijo directo de un div, cumple sólo la regla r1.
